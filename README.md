# Memo
사용자의 메모를 저장하고 수정하는 간단 메모 앱입니다 <br/>

# 개발 과정
AWS Lambda를 통해 Serverless 배포<br/> Visual Studio Code 로 서버 개발 및 API 개발<br/> PostMan 으로 API 테스트 <br/>
Android studio 를 통해 기능 구현 <br/>
JWT 토큰을 이용한 로그인 처리
